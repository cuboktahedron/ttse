テトリステンプレ検索
====

## どんなツールか
テトリス譜を管理して検索できます。

デモは[こちら](http://www.cuboktahedron.sakura.ne.jp/ttse/)

## 動作環境
- IE9以上
- Chrome
- Firefox
- 上記以外でもモダンなブラウザなら動作すると思います。
